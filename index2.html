FIR Complaint

Register criminal cases requiring police investigation

24-48h response
5 documents

Process Flow

  1. Visit police station with CNIC
  2. Provide complete incident details
  3. Submit evidence (photos, videos)
  4. Receive FIR copy with tracking number

Required Documents

  • Original CNIC
  • 2 passport photos
  • Affidavit (if required)
  • Evidence materials
  • Witness statements

CNIC Issues

Report lost/stolen national ID cards

Rs. 500 fee
7-10 days

Process Flow

  1. File police report immediately
  2. Get affidavit from notary
  3. Apply at NADRA office
  4. Submit police verification

Important

  • Report within 24 hours of loss
  • Block CNIC via NADRA helpline first
  • Carry original documents
